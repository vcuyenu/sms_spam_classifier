# SMS spam classifier 1.0
A simple ML model that uses feature extraction to classify SMSs into Spam (irrelevant messages) or Ham (relevant messages)
# Tools
Python, Pandas, Scikit-Learn
# Dataset 
SMS Spam Collection Dataset (UCI)
# Model
Multinomial Naive Bayes
# Metrics
Accuracy score, tester function
